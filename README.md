# AI Real Estate Underwriting Copilot

AI-powered investment analysis platform using LLMs, RAG, and agentic workflows for real estate financial decision-making.

This project demonstrates how modern AI systems can automate underwriting workflows including DSCR analysis, financial metric extraction, investment memo generation, and portfolio ranking.

## Key Capabilities

### AI Document Intelligence
Processes real estate financial documents including:

• rent rolls  
• operating statements  
• loan terms  
• construction budgets  
• property descriptions  

Extracts structured financial signals from unstructured documents.

---

### Retrieval-Augmented Generation (RAG)
Implements advanced retrieval pipeline:

• semantic chunking of financial documents  
• vector search using FAISS  
• metadata-aware retrieval  
• structured context building  

Ensures accurate and grounded AI responses.

---

### AI Underwriting Agent
Agent architecture performs financial reasoning:

• Net Operating Income (NOI) calculation  
• Debt Service Coverage Ratio (DSCR) computation  
• risk assessment  
• financial metric aggregation  
• investment comparison logic  

---

### Investment Memo Generator
Automatically produces structured investment summaries:

Executive summary  
key financial metrics  
risk interpretation  
investment recommendation  

Similar to internal credit memos used in real estate investment firms.

---

### Portfolio Ranking Engine
Ranks multiple properties based on:

DSCR strength  
expense efficiency  
rent stability  
lease duration  
tenant diversification  

Produces decision-ready ranking output.

---

### Streamlit Copilot Interface
Interactive UI for:

investment analysis queries  
memo generation  
portfolio analytics  
transparent evidence inspection  

Designed to resemble production-grade analytics software.
